    • CSS Introduction

    Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).

    CSS is designed to enable the separation of presentation and content, including layout, colors, fonts. This separation can improve content accessiblity;enable multiple web pages to share formatting by specifying the relevant CSS is a separate .css file, which reduces complexity and repetition in the structural content; and enable the .css file to be cached to improved the page load speed between the pages that share the file and its formatting.

    • CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
    • CSS has a simple syntax and uses a number of words from the English language to specify multiple properties.

    CSS Syntax

    A CSS rule consist of a selector and a declaration block. The selector points to the HTML element you want to style, while the declaration consist of a property and a value, separated by a colon. The multiple declarations are separated by semicolon.

    In the example below we are using CSS to change the color and font of an HTML p tag.

    p {color: #000000; font-style: bold;}

    In the declaration above we gave the paragraph element a default color of black and a bold font

    • Each declaration block is surrounded by curly braces.

    CSS Selectors

    A CSS selecto selects the HTML element(s) you wish to style. They are multiple selectors which are simple selectors, combinator selectors, pseudo-class selectors, pseudo-elements selectors and attribute selectors.

    • Simple Selectors - Select elements based on name, id, class.

    The example below shows three mentioned simple selectors in order.

    Name Selector
    p {margin: 30px auto; width: 100%;}
    ID Selector (#)
    #navbar {position: absolute; top: 0; left: 6em; bottom: 0;}
    Class Selector (.)
    .nav {background-color: #ffffff; position: fixed; top: 0;}
    • Combinator Selector - Selects elements based on a specific relationship between them

    The examples below shows three combinator selectors.

    Descendant selector (space)
    div p {background-color: rgb(0, 0, 255);}
    Child Selector (>)
    div > p {background-color: rgb(0, 0, 255);}
    Adjacent Sibiling Selector (+)
    div + p {background-color: rgb(0, 0, 255);}
    • Pseudo-Classes Selector - Selector element based on a certain state.

    The examples below shows three pseudo-classes selectors

    :hover Pseudo-Classes Selector
    a:hover {color: rgb(112, 56, 155);}
    :first-child Pseudo-Classes Selector
    a:first-child {color: rgb(112, 56, 155);}
    :last-of-type Pseudo-Classes Selector
    a:last-of-type {color: rgb(112, 56, 155);}
    • Pseudo-Elements Selector - Select and style a part of an element.

    The text below shows three psuedo-elements selectors.

    ::first-line Pseudo-element selector
    p::first-line {color: rgba(255, 255, 255, 70%);}
    ::first-letter Pseudo-element selector
    p::first-letter {color: rgba(255, 255, 255, 70%);}
    ::marker Pseudo-element selector
    p::marker {color: rgba(255, 255, 255, 70%);}
    • Attribute Selector - Selects an element based on an attribute or attribute value.

    The example below shows three attribute selectors.

    [attribute] Selector
    a[target] {background-color: #1a1a7b;}
    [attribute~="value"] Selector
    [title~="flower"] {border: 5px solid #1a1a7b;}
    [attribute|="value"] Selector
    [class|="top"] {background-color: #1a1a7b;}

    CSS Comments

    CSS comments are not displayed in the broswer, but they can help document the source code. they are used to explain the code, and may help when you edit the source code at a later date.

    • Comments are ignored by the browsers.
    • Comments can be added anywhere is a style element

    The example below shows two comment methods.

    Single Line Comment
    /* This is a single line comment*/
    p {font-style: bold;}
    Multiple Line Comment
    /* This is a comment
    on multiple
    lines */

    CSS Colors

    Colors are specified or called, using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values. This allows us to change the background color and text color of HTML elements.

    The example below shows three color values

    • This will change the li with a class of example-li
    RGB Color Value
    .example-li {background-color: rgb(0, 0, 255);}
    HEX Color Value
    .example-li {background-color: #1a1a7b;}
    HSL Color Value
    .example-li {background-color: hsl(100, 50%, 75%)}
